El presente proyecto corresponde a un sistema que permite la entrada mediante el teclado
de una expresión regular, la cual posteriormente es intervenida para generar un AFND, ese 
AFND sera interpretado en un archivo de extención "dot", el cual pasara por el software
Graphviz, software que permite transformar la información escrita en el archivo "dot",
llamado "Automata1" para el AFND, convirtiendola a una imagen ".jpg", para poder ser visualizada
con facilidad.
